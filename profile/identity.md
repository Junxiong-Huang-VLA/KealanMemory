# 身份定位（Identity）

## 角色定位

**独立全栈研究工程师**，一人承担研究 + 工程双轨工作，不依赖团队分工。

- 研究方向：计算机视觉、多模态理解、VLA（Vision-Language-Action）、具身智能
- 工程方向：实验室场景智能化、SOP 数字化合规系统、视觉检测工业应用
- 产品方向：AI 辅助实验记录、视频事件理解、时间线生成与证据链构建

## 核心技术方向

### 感知层
- 目标检测：YOLO 系列（当前主力 YOLO26s）
- 视频理解：帧采样 + 事件检测 + 时序分析
- 多模态：Qwen-VL / DashScope 图文理解
- 姿态估计：YOLOv8-pose / YOLO26s-pose

### 推理层
- 物理事件检测：hand-object interaction / liquid transfer / container state change 等
- 步骤匹配：SOP 协议图 + 事件证据链匹配
- 语义增强：VLM 驱动的素材命名与活动描述

### 工程层
- 后端：FastAPI + Python
- 前端：Vite + Vue/React
- 数据管理：SQLite + 向量检索
- 训练平台：本地 RTX 4050 / AutoDL RTX 5090

## 长期目标

1. 建立可复用的"实验室视觉理解基础设施"（感知→事件→步骤→合规）
2. 将视觉理解能力迁移至 VLA 闭环控制（感知→规划→执行）
3. 形成跨场景（化学实验室 / 工业检测 / 手术室等）的通用方法论
4. 输出工程系统 + 学术论文双轨成果

## 典型任务类型

| 任务类型 | 频率 | 典型产出 |
|---------|------|---------|
| 模型训练与评估 | 高 | 权重文件、评估报告 |
| 系统后端开发 | 高 | FastAPI 路由、处理链路 |
| 数据标注与管理 | 中 | Roboflow 数据集、dataset.yaml |
| 论文写作 | 中 | 方法描述、实验分析、图表 |
